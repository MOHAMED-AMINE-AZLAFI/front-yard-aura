# Front Yard Aura Image System

## Direction

All editorial imagery should feel like real DSLR photography of luxury American homes: natural light, authentic landscaping, realistic textures, and calm premium color.

## Requirements

- Default image ratio: Pinterest 2:3.
- Default production size: `1000x1500`.
- Use real photography sources, not AI-generated visuals.
- Avoid plastic textures, distorted homes, impossible plants, overprocessed contrast, and unrealistic lighting.
- Text overlays use white or beige typography, soft dark overlays, and restrained blur.

## Implementation

- Central image utilities: `src/data/images.ts`
- Overlay card component: `src/components/ImageOverlayCard.astro`
- MDX article image component: `src/components/mdx/ArticleImage.astro`

## Overlay Positions

- `top`
- `center`
- `bottom`
