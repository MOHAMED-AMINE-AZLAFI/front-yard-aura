# Front Yard Aura UI System

## Experience Direction

The interface is designed to feel like a premium American landscaping magazine: calm, editorial, visual, trustworthy, and fast.

## Core UI Rules

- Rounded corners use soft luxury radii, mostly `1rem` to `1.25rem`.
- Shadows are soft and low contrast, using `shadow-soft` and `shadow-premium`.
- Motion is subtle: hover lift, hover zoom, and smooth transitions.
- Navigation is sticky, glass-like, and mobile-first.
- Pinterest-style discovery uses masonry columns without JavaScript.
- Cards use image zoom areas, clear headlines, and restrained metadata.

## Utility Classes

- `btn-primary`
- `btn-secondary`
- `premium-card`
- `image-zoom`
- `section-band`
- `masonry-grid`
- `masonry-item`
- `link-premium`

## Performance

The design system is CSS-first and works with Astro SSG. Interactive polish is handled through CSS transitions so Core Web Vitals remain strong.
