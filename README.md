# Front Yard Aura

Static Astro site for a large SEO, Pinterest, and blog content operation focused on premium front yard landscaping ideas.

## Stack

- Astro SSG
- Tailwind CSS
- MDX content collections
- Static deploy targets: Cloudflare Pages, Netlify, Vercel

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Content

Articles live in `src/content/blog` as MDX files. The site currently includes 300 published MDX articles across the eight front yard landscaping categories.

## Deployment

Build command: `npm run build`

Output directory: `dist`

Cloudflare Pages, Netlify, and Vercel are configured for static deployment.
